# VS Code settings

## Sync settings
### Go to settings dir
MacOS VS Code
```bash
cd ~/Library/Application\ Support/Code/User
```
MacOS VS Code Insiders
```bash
cd ~/Library/Application\ Support/Code\ -\ Insiders/User
```

### Clone this repo
```bash
git init && git remote add origin https://github.com/sabahtalateh/vscode-settings.git
```
